# Pravigo::Duration

extends ActiveSupport::Duration class to add iso8601 support

## Installation

Add this line to your application's Gemfile:

    gem 'pravigo-duration'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pravigo-duration
